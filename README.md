# 上课形式
- 老师讲解2节课
- 学生分享的2节课：
    - 小组文献介绍和讨论1节课
    - 小组项目介绍和讨论1节课。

# 考核情况

- 到课情况（助教统计）、
- 每周阅读文献的presentation得分（任课老师打分）
- 小组研究项目海报的展示（学期末有一次集中的海报展示，老师集体打分）
- 最后的小组大项目论文
    - 可选项目：为了鼓励个人得分，除了提交小组项目之外，想得到高分的同学还可以提交一个人项目
